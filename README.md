# VehiculosElectricosAnalisis
Análisis exploratorio de 166K vehículos eléctricos en EE.UU. | Python · Pandas · Seaborn



Este proyecto presenta un análisis exploratorio de datos (EDA) sobre la población de vehículos eléctricos en Estados Unidos.
El objetivo es identificar patrones, relaciones y tendencias mediante técnicas de análisis univariado, bivariado y multivariado



Objetivo:
Analizar la distribución de vehículos eléctricos en función del tiempo
Comparar tipos de vehículos eléctricos (BEV vs PHEV)
Evaluar relaciones entre variables como rango eléctrico, año y precio
Identificar patrones relevantes para modelado futuro
